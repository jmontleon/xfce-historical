XFbdMgr

Figure 6-2. XFbdMgr

XFbdMgr is a little program that can manage lists of backdrops, from which xfbd can choose at startup (or when executed manually). Instead of a single picture you have to choose a list. There are example lists in the standard xfce backdrop directory.
