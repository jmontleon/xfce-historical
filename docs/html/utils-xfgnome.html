xfgnome

xfgnome is a xfwm module used for GNOME compatibility. If started, xfwm acts like a "GNOME compliant Window manager" and allow the use of GNOME applets such as GNOME pager.

This module can be activated from the Startup tabfolder in the setup dialog.
