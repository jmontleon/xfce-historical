You can associate a UNIX command to every icon on the Front Panel. You can even change the icons displayed, to suit your needs.

          1) The dialog box
           

          Enter the UNIX command to be executed when you click on the associated icon in the field "Command line".

          You can use :
           

          • Any UNIX command. XFCE starts a shell script before executing your command.
          • If your command begins with the keyword module, it will spawned to FVWM. Therefore, you can start any FVWM module from XFCE.

            • Example :  To start the module FvwmPager whenever you click on the button, use the following syntax :

            module FvwmPager 0 3
             

          You can also "browse" the commands or the icons by clicking on the corresponding button, and choose the command from the file selector :
