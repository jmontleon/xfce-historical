Vous pouvez changer le label de chacun des bureaux virtuels de XFCE.

          1) La boîte de dialogue
           
           
          Pour changer un label, il suffit de cliquer sur le bouton avec le bouton droit de la souris. Cela ouvira une boîte de dialogue dans laquelle il vous sera possible de changer le label. 
           
           

          Note : Les noms de chacun des bureaux virtuels est sauvegardé en même temps que le fichier de configuration.
