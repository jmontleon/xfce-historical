Il est possible d'associer une commande UNIX a chaque icône de la barre principale. Il est même possible de changer l'icône affichée, en fonction de vos besoins.

          1) La boîte de dialogue
           

           

          Entrez dans le champ "Commande" la commande UNIX a exécuter quand l'icône associée est activée.

          Il est possible d'utiliser :
           

          • N'importe quelle commande UNIX. XFCE démarre un shell en tâche de fond avant d'exécuter une commande.
          • Si la commande commence par le mot clé module, elle sera transmise à FVWM. De cette manière, il est possible de lancer des modules de FVWM depuis XFCE.

            • Exemple :
             

                module FvwmPager 0 3

                démarrera le module FvwmPager chaque fois que l'entrée sera sélectionnée dans XFCE.
