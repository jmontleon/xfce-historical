Le gestionnaire de fonds d'écran d'XFCE est un programme séparé utilisé pour afficher des images XPM dans la fenêtre racine.

          1) Démarrer XFBD
           

           

          XFBD procure une interface graphique pour parcourir et sélectionner le fond d'écran parmi les nombreux papiers peints livrés avec XFCE.
          Ce programme peut aussi être utilisé sans son interface, pour afficher automatiquement l'image XPM (X Pixmap) dans la fenêtre racine.
           

          • xfbd -d ou juste  xfbd seulement lit le nom du papier peint dans le fichier de configuration et l'affiche en fond d'écran.

            • Cette option doit être utilisée au démarrage d'une session, dans le script de démarrage comme .login ou .Xsession ou encore dans le fichier de configuration de FVWM (ce qui est le cas avec le fichier donné en exemple avec XFCE).
             
          • xfbd -i lance XFBD avec l'interface graphique montrée ci-dessus. Cette option doit être utilisée chaque fois que l'utilisateur (vous-même ?) décide de chager le papier peint de son écran.

            XFBD écrit ses données dans un fichier de configuration séparé, appelé .xfbdrc, placé dans le répertoire de l'utilisateur.

          Une fois démarré, XFBD recherche son fichier de configuration dans l'ordre suivant :
           

          • .xfbdrc dans le répertoire personnel de l'utilisateur.
          • .xfbdrc dans le répertoire /var/XFCE
          • sample.xfbdrc dans /var/XFCE
          2) Utilisation
           

          Il est possible de taper directement le nom du fichier image Pixmap, y compris son chemin d'accès complet, si vous le connaissez. Où alors, il est plus simple d'utiliser les boutons suivants :
           

          • "Ok" sauvegarde le fichier sélectionné et ferme XFBD.
          • "Load" permet de parcourir et de sélectionner le fichier image dans la liste des fichiers disponibles avec XFCE.
          • "Apply" applique le papier peint actuellement sélectionné.
          • "Clear" efface la sélection. Cette option est utilisée si vous ne voulez pas qu'XFBD n'affiche de papier peint (si vous avez choisit "Associer les couleurs au fond d'écran" dans les Paramètres de configuration, par exemple.)
          • "Cancel" ferme XFBD sans sauvegarder la sélection.
          Note : Il est à noter qu'XFBD consomme beaucoup de couleurs, en fonction du papier peint choisit, et il se peut donc qu'il ne soit pas souhaîtable sur des écrans disposant de moins de 65536 couleurs.
