XFCE nécessite la librairie XForms, version 0.81 ou supérieure. Il est nécessaire d'installer cette librairie AVANT de compiler ou d'utiliser XFCE.
           
          1) Où obtenir la librairie XForms
           
           Visitez http://bragg.phys.uwm.edu/xforms
           

          2) Téléchargez la version de la librairie correspondante à votre plateforme.

          Par exemple, si vous téléchargez la version pour Linux, vous obtiendrez un fichier qui ressemble à (en fonction de la version téléchargée, évidemment)
           

            bxform-linuxELF.tgz
          Ce fichier est une archive tar compressée avec GNU Zip, exactement comme la distribution de XFCE.

          3) Decompressez l'archive dans le répertoire de votre choix (/usr/src, par exemple)
           
           

            $ tar -xvzf bxform-linuxELF.tgz -C /usr/src
          Cela devrait créer un sous-répertoire  /usr/src/xforms

          4) Installer la librarie et les entêtes C

          Copier la librairie vers /usr/X11/lib :
           

            $ cp -ivd /usr/src/xforms/FORMS/libforms* /usr/X11/lib
          Copier les entêtes C vers /usr/X11/include/X11 :
           
            $ cp -ivd /usr/src/xforms/FORMS/forms.h /usr/X11/include/X11
          Il faut noter que le répertoire /usr/X11 peut être lié au système. Sur certains systèmes, cela pourrait être /usr/X11R6 ou /usr/X386. Bien sûr, dans ce cas replacez X11 par ce qui convient à votre système.

          5) Reconfigurer ld (le chargeur dynamique de librairies pour Linux)

          En tant que root, tapez :
           

            $ ldconfig -v
          Vous devriez trouver libforms dans la liste des librairies connues.
