Every user can choose to use XFCE or not. Although, the System Manager can also help by providing a default ".xinitrc" configuration file using XFCE as default.
           

          1) The automatic scripts

          XFCE 2.0.0 provides an easy and elegant way to setup XFCE as default desktop on a user account.

          Just log in, and type
           

            $ xfce_setup

          This will backup the .Xclients and .xinitrc files to a hidden directory, and replace original files with links to default XFCE files.

          If you wish to remove XFCE, and backout to previous configuration files, just use
           

            $ xfce_remove

          That's it !

          Well, if you want to know what XFCE really needs to startup, to make your own configurations files, keep reading this document, otherwise use xfce_setup and xfce_remove to setup and remove XFCE easily.

          2) Start the X session

          Once xfce_setup has been performed, just start X as usual (with startx, for example) and enjoy your new desktop environment !
