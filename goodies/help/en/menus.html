The Popup Menus of XFCE provide an easy way to launch applications.

          1) Opening a Popup Menu

          Clicking on the little arrow button , on the Front Panel,  will open a Popup Menu :

           
           
           
          If the option "Use tear-off menus" is activate, you can "detach" the menu by clicking on the little button at the bottom of the menu.

          This will create a new window for this menu, which will not close down when you select an item.

          2) Selecting an item

          To select an item, just click on the label :

          3) Adding a new item to a menu

          Click on the top of the window "Add icon..."

          This will open the following dialog box described in section "Add/change Menu Items"
           

           
          To modify the icons, or associate a command to an icon,click on the corresponding label using the left mouse button. This will open a similar dialog box that will let you change the desired parameters. See section "Add/change Menu Items" for more details.
