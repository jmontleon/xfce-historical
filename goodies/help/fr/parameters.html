XFCE procure une manière facile et pratique de configurer son bureau.

          1) La boîte de dialogue
           
          Cliquez sur cette icône, sur la  barre principale de XFCE, pour ouvrir la boîte de configuration. 
           

           
           
          • Selectionner un des 8 boutons colorés, et changez la valeur RVB (Rouge, Vert, Bleu) en déplaçant le curseur correspondant.

            • La palette  de couleur est sauvegardée automatiquement quand vous cliquez sur "Ok"
          • Une fois la palette de couleurs définies, il est possible de la sauvegarder sur disque en utilisant le bouton "Sauve...".
          • De la même manière, il est possible de charger une palette existante, grâce au bouton "Charger".
          • Cliquez sur "Défaut" pour retrouver les valeurs par défaut.
          La palette de couleurs est savegardée dans un fichier de configuration séparés (".xfcolors"), pour permettre à d'autres programmes de partager ce fichier.
           
          • Sélectionnez l'option "Associer les couleurs au fond d'écran" si vous désirez que la fenêtre principale soit de la même couleur que le bonton de l'écran virtuel. Cette option n'est pas compatible avec XFBD, le gestionnaire de fond d'écran de XFCE, étant donné que XFBD affiche des pixmaps dans la fenêtre principale...
           
          • Selectionnez "Utiliser les menus détachables" si vous désirez cette option. Les menus détachables peuvent présenter des comportements différents suivant le gestionnaire de fenêtre utilisé. Par défaut, cette option est activée.
